# For Lộc, Tài, Nam, Lạc
- To work on the agents write-up:
1. <code>git clone https://github.com/fungusdesu/cs486-project .</code> (this assumes the current directory is completely empty)
2. <code>cd requirements</code>
3. <code>latexmk -pvc -pdf main.tex</code>

- Compiled PDFs must be included as a binary in the Releases section of the repo with its version tag. Do not directly commit the PDFs.

- Within <code>/requirements</code> lies the latexmk config file <code>.latexmkrc</code>.
